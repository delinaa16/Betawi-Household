# BetaWi - Household Helper Booking Platform

## Overview

BetaWi is a modern, secure household helper booking platform that connects households with verified domestic workers (nannies, cooks, cleaners, etc.). The platform prioritizes accountability, security, and seamless communication between households and the BetaWi admin team.

## Key Features

### For Households
- **Easy Registration** - Multi-step signup with comprehensive accountability information collection
- **Browse Verified Helpers** - Filter helpers by role, availability, and ratings
- **Secure Booking System** - Request services with detailed booking forms
- **Direct Communication** - Chat directly with BetaWi admin for support and inquiries
- **Payment Management** - Multiple payment options (Telebirr, Chapa, Card, Cash)
- **Booking Management** - Track active and completed bookings
- **Rating & Reviews** - Rate helpers and provide feedback
- **Issue Reporting** - Report problems directly to admin
- **Contract Management** - Digital service agreements with terms and conditions
- **Profile Management** - View and manage household information

### Accountability Features
- **National ID & Kebele ID** - Government-issued identification verification
- **Address & Location** - Complete address information for verification
- **Emergency Contacts** - Multiple contact options for emergencies
- **Family Contact** - Additional contact information for accountability
- **Helper Profiles** - Complete information on all domestic workers
- **Service Contracts** - Digital agreements for all bookings
- **Issue Tracking** - Report and track any incidents

## How to Use

### For New Households
1. Click "Get Started" on the home page
2. Complete the 3-step registration process
3. Verify your email address
4. Browse available helpers
5. Select a helper and submit a booking request
6. Communicate with admin via chat
7. Complete payment
8. Rate and review after service

### For Existing Households
1. Log in with your credentials
2. View your dashboard
3. Browse helpers or manage existing bookings
4. Chat with admin for support
5. Make payments and track bookings

## Installation & Setup

### Requirements
- Node.js 18+
- npm or yarn

### Steps
1. Download the project ZIP file
2. Extract the files
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server
5. Open `http://localhost:3000` in your browser

### Using shadcn CLI
\`\`\`bash
npx shadcn-cli@latest init
npm run dev
\`\`\`
